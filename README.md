# Sistema de Delivery – Projeto em Java

Este projeto apresenta uma implementação simples de um sistema de delivery desenvolvido em **Java**, com o objetivo de aplicar e consolidar os conceitos fundamentais da **Programação Orientada a Objetos (POO)**.

A aplicação simula o funcionamento básico de um serviço de pedidos, permitindo registrar informações e visualizar pedidos realizados dentro do sistema. O foco principal do projeto é a organização do código e a modelagem do problema utilizando os princípios da orientação a objetos.

Durante o desenvolvimento, são explorados os **quatro pilares da POO**:

* **Encapsulamento:** proteção e controle do acesso aos dados do sistema.
* **Abstração:** representação simplificada de elementos do mundo real dentro do código.
* **Herança:** conceito estudado para reutilização e extensão de comportamentos entre estruturas do sistema.
* **Polimorfismo:** possibilidade de diferentes objetos responderem de maneiras distintas a uma mesma operação.

O objetivo deste projeto é **praticar a estruturação de aplicações utilizando boas práticas de programação orientada a objetos**, servindo como base para o desenvolvimento de sistemas mais complexos no futuro.
